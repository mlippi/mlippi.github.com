LAMP on Ubuntu-12.10

Install LAMP on Ubuntu-12.10:

$ sudo apt-get install tasksel
$ sudo tasksel
$ sudo apt-get install phpmyadmin
$ /etc/init.d/apache2 restart
$ cd /home/mlippi
$ mkdir www/test
$ sudo cp /etc/apache2/sites-available/default /etc/apache2/sites-available/test
$ ln -s /etc/apache2/sites-available/test /etc/apache2/sites-enabled/test
$ sudo mcedit /etc/apache2/sites-available/test
$ cd /home/mlippi

  ServerName test
  DocumentRoot /home/mlippi/www/test

  Options FollowSymLinks
  AllowOverride All


  Options Indexes FollowSymLinks MultiViews
  AllowOverride All
  Order allow,deny
  allow from all


$ sudo a2ensite test
$ sudo a2enmod rewrite
$ sudo mcedit /etc/hosts
127.0.0.2 test
$ sudo /etc/init.d/apache2 restart
